Paint-C#.Net clone
By Karim Oumghar
========

Small paint clone done in C#.Net
**Done so far**:

-Pen tool

-Paint brush

-Fill

-Clear drawing

-Eraser

-Brush thickness


**Currently still in progress...**

Undo/Redo, Edit menu, File menu, Drawing shapes, Editing text, Transformations, Selecting tool, Zoom
